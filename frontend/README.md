# Padel Club Frontend

React frontend aplikacija za padel klub.

## Funkcionalnosti

- pregled ponude terena, treninga, opreme i clanarine
- detaljna strana svake stavke
- dodavanje stavki u rezervacije
- unos podataka za termin
- pregled i potvrda rezervacije bez backend servera

## Pokretanje

```powershell
npm install
npm start
```
